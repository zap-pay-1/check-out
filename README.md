# 🛒 ZapPay Checkout

ZapPay Checkout is a streamlined payment module allowing users to make secure cross-chain payments directly from your site or app. It supports QR code payments, multi-chain options, and provides a developer-friendly API for easy integration.

---

## 🚀 Features

- 🔗 **Payment Link Generation**: Generate custom payment links for users.
- 🧾 **Invoice Checkout**: Quick and easy invoice payments.
- 📲 **QR Code Payments**: Mobile-friendly QR code scanning.
- 🌐 **Cross-Chain Flexibility**: Allow users to pay from any ERC20-compatible chain.

---

## 💻 Tech Stack

- **Node.js** and **Express.js**: Server-side handling
- **Particle Network**: Simplified wallet connections
- **Klaster**: Cross-chain abstraction for payments
- **REST API**: Standardized API for frontend-backend communication
